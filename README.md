## 구디아카데미69기 파이널프로젝트(Hanbando_tour)


### 소개
'한반도투어'는 패키지여행과 호텔예약이 가능한 여행사를 목적으로 진행된 프로젝트입니다.
'한반도투어'라는 이름은 모든 국내여행에 관련된 상품을 제공하자는 취지로 정해졌으며 회원을  대상으로 서비스를 제공합니다.
